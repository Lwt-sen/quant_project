# quant_project
量化数据支持系统：包含行情数据同步、数据质量校验、均线策略回测、IC/IR因子分析、可视化报告全链路
# 量化数据支持系统
基于 Python + MySQL 构建的工业级量化数据中台与策略回测系统，覆盖行情数据同步、数据质量治理、策略回测、因子绩效分析、可视化报告全链路。

##  项目架构
quant_project
.env # 环境变量（敏感配置）
config.py # 全局参数配置
db_manager.py # 数据库 ORM 封装 + 原子 Upsert 写入
sync_status.py # 增量同步状态管理（断点续传）
data_quality.py # 数据质量校验体系
data_loader.py # 行情数据加载 + 因子计算
strategy.py # 均线多股票调仓策略
backtest_engine.py # 回测引擎封装
factor_analysis.py # IC/IR/ 阿尔法 / 分位数因子分析
visualization.py # Plotly 交互式可视化报告
monitor.py # 同步运维监控与报告
main.py # 主程序入口

##  核心特性
- **数据工程化**：Tushare 行情增量同步 + 断点续传，MySQL Upsert 原子写入，唯一键保障数据一致性
- **质量管控体系**：多维度数据校验（空值、价格逻辑、涨跌幅异常），从源头拦截脏数据
- **模块化策略回测**：均线交叉多股票策略，支持持仓上限、定期调仓、三级止盈止损风控
- **专业因子分析**：IC/IR 计算、分位数收益检验、CAPM 阿尔法归因、收益分布统计
- **可观测性**：全链路结构化日志、同步运维报告、Plotly 交互式可视化分析
- **分层架构设计**：原始数据层 / 衍生因子层 / 策略层 / 分析层解耦，易扩展易维护

##  核心模块说明
### 1. 数据层
- 支持全量/增量两种同步模式，日常增量更新效率提升90%以上
- 基于唯一键的 Upsert 写入，重复运行不产生脏数据
- 连接池保活机制，适配生产环境长时运行

### 2. 数据质量治理
- 必填字段非空校验、数值非负校验
- 价格逻辑校验（最高价≥开/收≥最低价）
- 涨跌幅异常阈值告警，拦截异常数据
- 支持强校验/弱告警两种模式

### 3. 策略与回测
- 均线交叉选股逻辑，每10个交易日定期调仓
- 最多持有10只股票，等权分配资金
- 三级风控：固定止损 + 固定止盈 + 移动止盈
- 标准回测指标：年化收益、最大回撤、夏普比率、胜率

### 4. 因子绩效分析
- **IC/IR**：量化因子预测能力与稳定性
- **分位数检验**：验证因子收益单调性，排除极端值干扰
- **阿尔法归因**：基于CAPM模型拆解超额收益
- **收益分布分析**：中位数、偏度、峰度，评估收益结构健康度
